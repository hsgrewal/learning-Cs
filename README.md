# Learning-C#
This repository documents a journey to learn the basics of C# using
Visual Studio Code on a Linux (Ubuntu) platform.

## Installing VS code
Download the `.deb` from [here](https://code.visualstudio.com/download)

Follow the instructions provided
[here](https://code.visualstudio.com/docs/setup/linux)
to setup VS Code on Ubuntu.

## Configure VS Code for C# development
In the **Extensions** view (`Ctrl+Shift+X`) of VS Code, install a C# extension.

### Install .NET Core SDK
Download and install .NET Core SDK from
[here](https://www.microsoft.com/net/download)

## Test Configuration
Follow the .NET Tutorial found
[here](https://www.microsoft.com/net/learn/get-started-with-dotnet-tutorial)
to verify configuration for C# development.
